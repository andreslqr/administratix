Button

This is a Laravel and Livewire Implementation, to see more about the css classes please see Daisy Buttons

Attribute Type Values Description Default
content String Use instead of default laravel slot components null
loading Boolean true, false Show the loading state of the button false
ghost Boolean true, false Show button in ghost state false
as-link Boolean true, false Show the button as a anchor tag false
no-animation Boolean true, false Remove all the animations false
active Boolean true, false Show as it was clicked false
outline Boolean true, false Show the button with borders and without bakground color
wide Boolean true, false More horizontal padding false
glass Boolean true, false Glass effect false
disabled Boolean true, false Show the disabled state false
square Boolean true, false Show as square (recommended for icons) false
circle Boolean true, false Show as circle (recommended for icons) false
block Boolean true, false Use the full width false
text-loading String The text showed when is in loading state false
