HTML general inputs

This is a Laravel and Livewire Implementation, and this docs are only the default html inputs and the general configurations from DaisyUI

Checkbox

to see more about the css classes please see Daisy Checkbox

Attribute Type Values Description Default
Without available attributes
Preview Code 
            <x-admin::form.input.checkbox />

File input

to see more about the css classes please see Daisy File Input

Attribute Type Values Description Default
bordered Boolean true, false Show the input without borders false
ghost Boolean true, false Show the input as ghost state false
Preview Code 
            <x-admin::form.input.file />

Radio

to see more about the css classes please see Daisy Radio Input

Attribute Type Values Description Default
Without available attributes
Preview Code 
            <x-admin::form.input.radio wire:model="selected" :value="1" />
            <x-admin::form.input.radio wire:model="selected" :value="2" />

Range

to see more about the css classes please see Daisy Range Input

Attribute Type Values Description Default
min Integer The min value (html default behavior)
max Integer The max value (html default behavior)
step Integer The step value (html default behavior)
with-steps Boolean true, false Show lines under the range input false
Preview Code
| | | | | 
            <x-admin::form.input.range :min="0" :max="100" :step="25" with-steps />

Rating input

to see more about the css classes please see Daisy Rating Input

Attribute Type Values Description Default
name String The name of the input (use instead of wire:model)
values Array [...] The values of each rate (default 0 to max property) [0, 1, 2, 3, ...]
max Integer The max value for the generator of rate (doesn't work if you already use values property)
mask-class String The mask crop for the elements mask-star
Preview Code 
            <x-admin::form.input.rating :max="5" />
