Card

This is a Laravel and Livewire Implementation, to see more about the css classes please see Daisy Cards

Attribute Type Values Description Default
content String Use instead of default laravel slot components
top-image String Use as slot for the image top
bottom-image String Use as slot for the image bottom
actions String Use as slot for the actions as buttons
compact Boolean true, false Reduce the padding of the card false
image-on-side Boolean true, false Move the top-image to the left or the bottom-image to the right false
glass Boolean true, false Glass effect false
full-image Boolean true, false Show the image as background false
