Attribute Type Values Description Default
compact Boolean true, false Show small menu false
horizontal Boolean true, false Show menu in horizontal false
rounded Boolean true, false Round the corners of the menu false
content String Use instead of default slot null
active Boolean true, false Show an item as active false
as-title Boolean true, false Show the item as title limiter false
bordered Boolean true, false Show the item with rounded corners false
bordered-on-hover Boolean true, false Round corners on hover false
disabled Boolean true, false Show item as inactive false
anchor-class String true, false Extra classes for anchor element null
