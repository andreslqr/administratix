Attribute Type Values Description Default
id String Required to work without Livewire
content String Use as slot for the hidden content null
actions String Use as slot for the action section of the modal, like buttons null
with-close Boolean Show an X button for close false
static Boolean You can't close the modal clicking outside false

Controlling from Livewire

You can control any modal of your form from livewire if you add the \Administratix\Administratix\Support\Livewire\Components\WithModals trait in your component, then you can access to any method

Preview Code Livewire

        <x-admin::button wire:click="showModal('myModal')">open modal</x-admin::button>
        <x-admin::modal wire:modal="myModal">
            <x-slot name="content">
                <h3 class="font-bold text-lg">Congratulations random Internet user!</h3>
                <p class="py-4">You've been selected for a chance to get one year of subscription to use Wikipedia for free!</p>
                <x-slot name="actions">
                    <label for="my-modal" class="btn">Yay!</label>
                </x-slot>
            </x-slot>
        </x-admin::modal>
    
        <?php

        class MyLivewireComponent extends Component
        {
            use \Administratix\Administratix\Support\Livewire\Components\WithModals;
            
            /**
            * The initial state of the modals elements
            * 
            * @var array
            */
            public $modals = [
                'myModal' => false
            ];

            public function editData()
            {
                $this->toggleModal('myModal'); // Toggle the modal open / close

                $this->showModal('myModal'); // Show the modal

                $this->hideModal('myModal'); // Hide the modal

                $this->getModal('myModal'); // Return true or false depending if the modal is open or not

                if($this->isModalOpen('myModal')) // Check if the modal is open
                    $this->doSomething();

                if($this->isModalClosed('myModal')) // Check if the modal is closed
                    $this->doSomenthing();
            }
        }
