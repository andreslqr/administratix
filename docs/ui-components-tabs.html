Tabs

This is a Laravel and Livewire Implementation, to see more about the css classes please see Daisy Tabs

Attribute Type Values Description Default
tabs String Slot for tabs
content String Slot for content
items array The auto items/tabs array values null
bordered Boolean true, false Show border in the tabs false
lifted Boolean true, false Show lifted false
boxed Boolean true, false Show as button false
active-tab String Set the active tab via attribute null
for String The name for the tab
active Boolean true, false Show as active tab false

Controlling from Livewire

You can control any tabs of your form from livewire if you add the \Administratix\Administratix\Support\Livewire\Components\WithTabs trait in your component, then you can access to any method, the $tabs property defines al the tabs instance an the initial state on load

        <x-admin::tabs wire:tabs="menu" :items="['tab1' => 'Tab 1', 'tab2' => 'Tab 2', 'tab3' => 'Tab 3']">
            <x-slot name="contentTab1">
                Content for Tab 1
            </x-slot>
            <x-slot name="contentTab2">
                Content for Tab 2
            </x-slot>
            <x-slot name="contentTab3">
                Content for Tab 3
            </x-slot>
        </x-admin::tabs>
        
    
        <?php

        class MyLivewireComponent extends Component
        {
            use \Administratix\Administratix\Support\Livewire\Components\WithTabs;
            
            /**
            * The tab elements
            * 
            * @var array
            */
            public $tabs = [
                'menu' => 'tab2'
            ];


            public function editData()
            {
                $this->getTab('menu'); // get the current active tab

                $this->showTab('menu', 'tab2'); // set the active tab


                if($this->isTabOpen('menu', 'tab3')) // Check if the tab is open
                    $this->doSomething();

                if($this->isTabClosed('menu', 'tab1')) // Check if the tab is closed
                    $this->doSomenthing();
            }
        }
