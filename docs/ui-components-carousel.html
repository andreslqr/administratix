Attribute Type Values Description Default
items String Use instead of default laravel slot components null
center Boolean true, false Snap items to center false
end Boolean true, false Snap items to end false
vertical Boolean true, false Display as vertical false

Controlling from Livewire

You can control any carousel of your form from livewire if you add the \Administratix\Administratix\Support\Livewire\Components\WithCarousels trait in your component, then you can access to the method

Preview Code Livewire
CLICK 
        <x-admin::carousel class="w-64" wire:carousel="mainCarousel">
            @foreach($images as $name => $image)
                <x-admin::carousel.item>
                    <img src="\{{ $image }}" />
                </x-admin::carousel.item>
            @endforeach
        </x-admin::carousel>

        <x-admin::button wire:click="moveCarousel">
            CLICK
        </x-admin::button>
        
    
        <?php

        class MyLivewireComponent extends Component
        {
            use \Administratix\Administratix\Support\Livewire\Components\WithDropdowns;

            public function moveCarousel()
            {
                 $this->goToCarouselItem('mainCarousel', 1); 
            }
        }
