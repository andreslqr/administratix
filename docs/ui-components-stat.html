Stat

This is a Laravel and Livewire Implementation, to see more about the css classes please see Daisy Stats

Attribute Type Values Description Default
title String Slot for the title of the stat null
title String Slot for the title of the stat null
value String Slot for the stat value null
description String Slot for small description of the stat null
figure String Slot for an image/video figure null
actions String Slot for actions of the stat as buttons null
no-auto-wrap Boolean true, false Property of the wrapper to tell sthe child elements how to self display false
vertical Boolean true, false Property of the wrapper show stats in vertical mode null
