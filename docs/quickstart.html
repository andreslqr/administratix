Quickstart

Install Administratix

Install the php dependency 
composer require administratix/administratix
Then run the artisan installer 
php artisan administratix:install

Using the layout

After this, you can use the layouts 

<x-admin::layouts.app>
    <!-- your content here -->
</x-admin::layouts.app>

Adding routes

To add routes in the sidebar, use the Route::admin() facade into your routes file routes/web.php 

<?php

Route::admin(
    method: 'GET',
    uri: '/',
    action: function() {
        return view('index');
    },
    display: 'Home',
    iconComponent: 'admin::icon.awesome',
    iconName: 'home'
);
